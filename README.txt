YÊU CẦU:
-- Trước khi khởi tạo dự án
	- phiên bản của PHP trên máy > 8.0
	- đã cài đặt composer trên máy

PHẦN HƯỚNG DẪN:
-- Khi nhận giải nén được mục source code sẽ bao gồm 1 folder tổng chứa toàn bộ code
-- Di chuyển vào bên trong thư mục vừa được giải nén bằng cách:
	- cd movie-booking
-- sau khi đã vào thư mục source code 
 	- chạy câu lệnh : "composer update" để cập nhật lại những vendor kéo về hổ trợ cho việc debug và phát triển tính năng vd: "PHPMailer"
-- sau khi đã chạy xong câu lệnh "composer update"
	- mở thư mục với đường dẫn "http://localhost/movie-booking" sau khi đã đặt thư mục source nằm trong thư mục htdocs
-- tuỳ chỉnh lại db_name và mật khẩu của database trong file .env
